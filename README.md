# Calculator

A fast, modern calculator app built for desktop and mobile.

## What it does
- Basic arithmetic: addition, subtraction, multiplication, division
- Percent calculations (%)
- Parentheses support for grouped expressions
- Clear (AC) and delete (DEL) controls
- Quick, readable display with auto-resizing text for long results
- Calculation history (tap a result to reuse it)
- Settings panel for simple preferences (ex: vibration, sounds, auto-resize font)
- Smooth UI dialogs for notices and updates
- Mobile-friendly touch support

## Notes
This project focuses on a clean UI and a smooth calculator experience across devices.
